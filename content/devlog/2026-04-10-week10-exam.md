## What was built

A RESTful Java backend for a gamified coding learning platform. Users register, get a JWT token, and can browse courses, lessons, exercises and track progress. A Stack Overflow integration serves the most upvoted Java questions as supplementary learning material.

## Reflection

Nine weeks of solo backend development. Every layer was built from scratch — entities, DAOs, services, controllers, security, testing, deployment. The commit history tells the story: one layer per commit, nothing skipped.
