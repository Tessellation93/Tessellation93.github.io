First portfolio Friday of the semester. The task was to brainstorm 2-3 project ideas ahead of Monday's review with the teacher.

This week covered Java Threads and ExecutorService as the first topics of our 3rd semester. The portfolio project needs to touch all the semester's technologies as they're introduced, so the idea needs to be flexible enough to grow.

After considering a few directions, I landed on a coding learning platform something that addresses a gap I've noticed personally. Most resources for learning to code are either dense or disengaging. The idea is to build something rewarding, structured around Java with room to expand as new technologies are added each week.

Working title: JavaJolt. Core concept: a backend API for a gamified Java learning platform with exercises, progress tracking, user accounts, and eventually authentication and deployment.

**For Monday:** presenting the idea with pros/cons and an initial scope.
